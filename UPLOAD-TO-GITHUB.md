# One-time GitHub Pages setup

Repository:

`zarc70/Multi-Vehicle-Racer-Gdevelop-backend-`

## Upload the game

1. Open the repository on GitHub.
2. Select **Add file → Upload files**.
3. Unzip this package on your computer.
4. Drag **the contents of the unzipped folder** into GitHub.
   - Upload `index.html`, `game.js`, `manifest.webmanifest`, `service-worker.js`, `.nojekyll`, the `assets` folder, and the `icons` folder.
   - Do not upload the enclosing folder as another nested folder.
5. Commit the files directly to the `main` branch.

## Turn on GitHub Pages

1. Open **Settings** in the repository.
2. Select **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Choose:
   - Branch: `main`
   - Folder: `/ (root)`
5. Select **Save**.

GitHub will publish the game at:

`https://zarc70.github.io/Multi-Vehicle-Racer-Gdevelop-backend-/`

Publishing may take a few minutes the first time.

## Install it on iPhone

1. Open the published address in Safari.
2. Turn the iPhone sideways.
3. Tap **Share → Add to Home Screen**.
4. Open **Vehicle Racer** from the Home Screen.

## Future updates

Upload the contents of each new GitHub Pages-ready release over the existing repository files. The same public address will update automatically.
